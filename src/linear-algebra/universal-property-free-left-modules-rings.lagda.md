## Idea

Given a [left module](linear-algebra.left-modules-rings.md) `M` over a
[ring](ring-theory.rings.md) `R`, a type `G`, and a map `i : G → M`, the
{{#concept "universal property of free left modules" Disambiguation="over a ring" Agda=is-free-left-module-Ring}}
says that `M` is freely generated by `G` if for any left module `N` over `R`,
the mapping on [linear maps](linear-algebra.linear-maps-left-modules-rings.md)
from `M` to `N`, `φ ↦ φ ∘ i`, is an [equivalence](foundation.equivalences.md).

## Definition

```agda
module _
  {l1 l2 l3 : Level}
  (R : Ring l1)
  (G : UU l2)
  (M : left-module-Ring l3 R)
  (in-M : G → type-left-module-Ring R M)
  where

  is-free-left-module-Ring : UUω
  is-free-left-module-Ring =
    {l4 : Level} (N : left-module-Ring l4 R) →
    is-equiv (λ φ → map-linear-map-left-module-Ring R M N φ ∘ in-M)
```

## Properties

### The canonical linear map between free left modules

```agda
module _
  {l1 l2 l3 l4 : Level}
  (R : Ring l1)
  (G : UU l2)
  (M : left-module-Ring l3 R)
  (N : left-module-Ring l4 R)
  (in-M : G → type-left-module-Ring R M)
  (in-N : G → type-left-module-Ring R N)
  (is-free-M : is-free-left-module-Ring R G M in-M)
  (is-free-N : is-free-left-module-Ring R G N in-N)
  where

  linear-map-is-free-left-module-Ring : linear-map-left-module-Ring R M N
  linear-map-is-free-left-module-Ring =
    map-inv-is-equiv (is-free-M N) in-N
```

### The composition of the canonical linear maps in each direction between free left modules is the identity

```agda
module _
  {l1 l2 l3 l4 : Level}
  (R : Ring l1)
  (G : UU l2)
  (M : left-module-Ring l3 R)
  (N : left-module-Ring l4 R)
  (in-M : G → type-left-module-Ring R M)
  (in-N : G → type-left-module-Ring R N)
  (is-free-M : is-free-left-module-Ring R G M in-M)
  (is-free-N : is-free-left-module-Ring R G N in-N)
  where

  abstract
    is-section-linear-map-is-free-left-module-Ring :
      comp-linear-map-left-module-Ring R N M N
        ( linear-map-is-free-left-module-Ring R G M N in-M in-N
          ( is-free-M)
          ( is-free-N))
        ( linear-map-is-free-left-module-Ring R G N M in-N in-M
          ( is-free-N)
          ( is-free-M)) ＝
      id-linear-map-left-module-Ring R N
    is-section-linear-map-is-free-left-module-Ring =
      is-injective-is-equiv
        ( is-free-N N)
        ( eq-htpy
          ( ( ( ap
                ( map-linear-map-left-module-Ring R M N
                  ( map-inv-is-equiv (is-free-M N) in-N))) ∘
              ( htpy-eq (is-section-map-inv-is-equiv (is-free-N M) in-M))) ∙h
            ( htpy-eq (is-section-map-inv-is-equiv (is-free-M N) in-N))))
```

### The canonical isomorphism between any two free left modules

```agda
module _
  {l1 l2 l3 l4 : Level}
  (R : Ring l1)
  (G : UU l2)
  (M : left-module-Ring l3 R)
  (N : left-module-Ring l4 R)
  (in-M : G → type-left-module-Ring R M)
  (in-N : G → type-left-module-Ring R N)
  (is-free-M : is-free-left-module-Ring R G M in-M)
  (is-free-N : is-free-left-module-Ring R G N in-N)
  where

  is-iso-linear-map-is-free-left-module-Ring :
    is-iso-left-module-Ring R M N
      ( linear-map-is-free-left-module-Ring R G M N in-M in-N
        ( is-free-M)
        ( is-free-N))
  is-iso-linear-map-is-free-left-module-Ring =
    ( linear-map-is-free-left-module-Ring R G N M in-N in-M
        ( is-free-N)
        ( is-free-M) ,
      is-section-linear-map-is-free-left-module-Ring R G M N in-M in-N
        ( is-free-M)
        ( is-free-N) ,
      is-section-linear-map-is-free-left-module-Ring R G N M in-N in-M
        ( is-free-N)
        ( is-free-M))

  iso-is-free-left-module-Ring : iso-left-module-Ring R M N
  iso-is-free-left-module-Ring =
    ( linear-map-is-free-left-module-Ring R G M N in-M in-N
        ( is-free-M)
        ( is-free-N) ,
      is-iso-linear-map-is-free-left-module-Ring)
```

### If a free module is isomorphic to another module, that module is free

```agda
module _
  {l1 l2 l3 l4 : Level}
  (R : Ring l1)
  (G : UU l2)
  (M : left-module-Ring l3 R)
  (in-M : G → type-left-module-Ring R M)
  (is-free-M : is-free-left-module-Ring R G M in-M)
  (N : left-module-Ring l4 R)
  (iso-M-N : iso-left-module-Ring R M N)
  where

  is-free-iso-left-module-Ring :
    is-free-left-module-Ring R G N
      ( map-iso-left-module-Ring R M N iso-M-N ∘ in-M)
  is-free-iso-left-module-Ring P =
    is-equiv-comp _ _
      ( is-equiv-precomp-hom-iso-Large-Precategory
        ( large-precategory-left-module-Ring R)
        ( iso-M-N)
        ( P))
      ( is-free-M P)
```
