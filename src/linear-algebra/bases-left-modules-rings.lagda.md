## Idea

A family of elements `mᵢ` for `i : I` of a
[left module](linear-algebra.left-modules-rings.md) `M` over a
[ring](ring-theory.rings.md) `R` is a
{{#concept "basis" WDID=Q810193 WD="basis" Disambiguation="of a left module over a ring"}}
if `M` is
[freely generated](linear-algebra.universal-property-free-left-modules-rings.md)
by the `mᵢ`. However, since this is not a small
[proposition](foundation.propositions.md), we use an equivalent substitute.

Let `FI` be a module over `R` freely generated by `I` (using the
[canonical construction](linear-algebra.freely-generated-left-modules-rings.md)).
By the universal property of freely generated modules, the map `i ↦ mᵢ` induces
a [linear map](linear-algebra.linear-maps-left-modules-rings.md) `φ : FI → M`.
`M` is freely generated by the `mᵢ`
[if and only if](foundation.logical-equivalences.md) `φ` is an
[isomorphism](linear-algebra.isomorphisms-left-modules-rings.md). Being an
isomorphism is a small proposition (in the smallest universe we could plausibly
hope for) and we therefore use it as our definition.

## Definition

### The property that an indexed family of elements of a left module is a basis

```agda
module _
  {l1 l2 l3 : Level}
  (R : Ring l1)
  (M : left-module-Ring l2 R)
  (I : UU l3)
  (basis : I → type-left-module-Ring R M)
  where

  is-indexed-basis-prop-left-module-Ring : Prop (l1 ⊔ l2 ⊔ l3)
  is-indexed-basis-prop-left-module-Ring =
    is-iso-prop-left-module-Ring
      ( R)
      ( free-left-module-Ring R I)
      ( M)
      ( map-inv-is-equiv (is-free-free-left-module-Ring R I M) basis)

  is-indexed-basis-left-module-Ring : UU (l1 ⊔ l2 ⊔ l3)
  is-indexed-basis-left-module-Ring =
    type-Prop is-indexed-basis-prop-left-module-Ring
```

### The type of bases at a specified universe level

```agda
module _
  {l1 l2 : Level}
  (l3 : Level)
  (R : Ring l1)
  (M : left-module-Ring l2 R)
  where

  basis-left-module-Ring : UU (l1 ⊔ l2 ⊔ lsuc l3)
  basis-left-module-Ring =
    Σ ( UU l3)
      ( λ I → type-subtype (is-indexed-basis-prop-left-module-Ring R M I))

module _
  {l1 l2 l3 : Level}
  (R : Ring l1)
  (M : left-module-Ring l2 R)
  ((I , b , is-basis-b) : basis-left-module-Ring l3 R M)
  where

  type-index-basis-left-module-Ring : UU l3
  type-index-basis-left-module-Ring = I

  index-basis-left-module-Ring :
    type-index-basis-left-module-Ring → type-left-module-Ring R M
  index-basis-left-module-Ring = b

  is-indexed-basis-basis-left-module-Ring :
    is-indexed-basis-left-module-Ring R M
      ( type-index-basis-left-module-Ring)
      ( index-basis-left-module-Ring)
  is-indexed-basis-basis-left-module-Ring = is-basis-b
```

## Properties

### An indexed family of elements is a basis if and only if the module is freely generated by those elements

```agda
module _
  {l1 l2 l3 : Level}
  (R : Ring l1)
  (M : left-module-Ring l2 R)
  (I : UU l3)
  (basis : I → type-left-module-Ring R M)
  where

  abstract
    is-free-is-indexed-basis-left-module-Ring :
      is-indexed-basis-left-module-Ring R M I basis →
      is-free-left-module-Ring R I M basis
    is-free-is-indexed-basis-left-module-Ring is-basis-basis N =
      tr
        ( is-equiv)
        ( ap
          ( λ f φ → map-linear-map-left-module-Ring R M N φ ∘ f)
          ( is-section-map-inv-is-equiv
            ( is-free-free-left-module-Ring R I M)
            ( basis)))
        ( is-free-iso-left-module-Ring R I
          ( free-left-module-Ring R I)
          ( in-free-left-module-Ring R I)
          ( is-free-free-left-module-Ring R I)
          ( M)
          ( map-inv-is-equiv (is-free-free-left-module-Ring R I M) basis ,
            is-basis-basis)
          ( N))

    is-indexed-basis-is-free-left-module-Ring :
      is-free-left-module-Ring R I M basis →
      is-indexed-basis-left-module-Ring R M I basis
    is-indexed-basis-is-free-left-module-Ring =
      is-iso-linear-map-is-free-left-module-Ring
        ( R)
        ( I)
        ( free-left-module-Ring R I)
        ( M)
        ( in-free-left-module-Ring R I)
        ( basis)
        ( is-free-free-left-module-Ring R I)
```
