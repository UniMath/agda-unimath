## Idea

A family of elements `vᵢ` for `i : I` of a
[vector space](linear-algebra.vector-spaces.md) `V` over a
[Heyting field](commutative-algebra.heyting-fields.md) `K` is a
{{#concept "basis" WDID=Q189569 WD="basis" Disambiguation="of a vector space"}}
if `V` is
[freely generated](linear-algebra.universal-property-free-left-modules-rings.md)
by the `vᵢ`. However, since this is not a small
[proposition](foundation.propositions.md), we use an equivalent substitute.

Let `FI` be a vector space over `K` freely generated by `I` (using the
[canonical construction](linear-algebra.freely-generated-vector-spaces.md)). By
the universal property of freely generated vector spaces, the map `i ↦ vᵢ`
induces a [linear map](linear-algebra.linear-maps-left-modules-rings.md)
`φ : FI → V`. `V` is freely generated by the `Vᵢ`
[if and only if](foundation.logical-equivalences.md) `φ` is an
[isomorphism](linear-algebra.isomorphisms-vector-spaces.md). Being an
isomorphism is a small proposition (in the smallest universe we could plausibly
hope for) and we therefore use it as our definition.

## Definition

```agda
module _
  {l1 l2 l3 : Level}
  (K : Heyting-Field l1)
  (V : Vector-Space l2 K)
  (I : UU l3)
  (basis : I → type-Vector-Space K V)
  where

  is-indexed-basis-prop-Vector-Space : Prop (l1 ⊔ l2 ⊔ l3)
  is-indexed-basis-prop-Vector-Space =
    is-indexed-basis-prop-left-module-Ring
      ( ring-Heyting-Field K)
      ( V)
      ( I)
      ( basis)

  is-indexed-basis-Vector-Space : UU (l1 ⊔ l2 ⊔ l3)
  is-indexed-basis-Vector-Space = type-Prop is-indexed-basis-prop-Vector-Space
```

### The type of bases at a universe level

```agda
module _
  {l1 l2 : Level}
  (l3 : Level)
  (K : Heyting-Field l1)
  (V : Vector-Space l2 K)
  where

  basis-Vector-Space : UU (l1 ⊔ l2 ⊔ lsuc l3)
  basis-Vector-Space = basis-left-module-Ring l3 (ring-Heyting-Field K) V

module _
  {l1 l2 l3 : Level}
  (K : Heyting-Field l1)
  (V : Vector-Space l2 K)
  (b : basis-Vector-Space l3 K V)
  where

  type-index-basis-Vector-Space : UU l3
  type-index-basis-Vector-Space =
    type-index-basis-left-module-Ring (ring-Heyting-Field K) V b

  index-basis-Vector-Space :
    type-index-basis-Vector-Space → type-Vector-Space K V
  index-basis-Vector-Space =
    index-basis-left-module-Ring (ring-Heyting-Field K) V b
```

## Properties

### An indexed family of elements is a basis if and only if the vector space is freely generated by those elements

```agda
module _
  {l1 l2 l3 : Level}
  (K : Heyting-Field l1)
  (V : Vector-Space l2 K)
  (I : UU l3)
  (basis : I → type-Vector-Space K V)
  where

  abstract
    is-free-is-indexed-basis-Vector-Space :
      is-indexed-basis-Vector-Space K V I basis →
      is-free-Vector-Space K I V basis
    is-free-is-indexed-basis-Vector-Space =
      is-free-is-indexed-basis-left-module-Ring (ring-Heyting-Field K) V I basis

    is-indexed-basis-is-free-Vector-Space :
      is-free-Vector-Space K I V basis →
      is-indexed-basis-Vector-Space K V I basis
    is-indexed-basis-is-free-Vector-Space =
      is-indexed-basis-is-free-left-module-Ring (ring-Heyting-Field K) V I basis
```
