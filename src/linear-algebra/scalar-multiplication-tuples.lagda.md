## Idea

Any operation `B → A → A` for some type `B` of formal scalars induces an
operation `B → vec n A → vec n A`.

## Definition

```agda
scalar-mul-vec :
  {l1 l2 : Level} {B : UU l1} {A : UU l2} {n : ℕ} →
  (B → A → A) → B → vec A n → vec A n
scalar-mul-vec μ x = map-vec (μ x)
```
