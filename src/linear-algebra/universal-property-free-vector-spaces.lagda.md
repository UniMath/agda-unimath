## Idea

Given a [vector space](linear-algebra.vector-spaces.md) `V` over a
[Heyting field](commutative-algebra.heyting-fields.md) `K`, a type `G`, and a
map `i : G → V`, the
{{#concept "universal property of free vector spaces" Agda=is-free-Vector-Space}}
says that `V` is freely generated by `G` if for any vector space `W` over `K`,
the mapping on [linear maps](linear-algebra.linear-maps-vector-spaces.md) from
`V` to `W`, `φ ↦ φ ∘ i`, is an [equivalence](foundation.equivalences.md).

## Definition

```agda
is-free-Vector-Space :
  {l1 l2 l3 : Level}
  (K : Heyting-Field l1)
  (G : UU l2)
  (V : Vector-Space l3 K) →
  (G → type-Vector-Space K V) →
  UUω
is-free-Vector-Space K = is-free-left-module-Ring (ring-Heyting-Field K)
```

## Properties

### The canonical linear map between free vector spaces

```agda
module _
  {l1 l2 l3 l4 : Level}
  (K : Heyting-Field l1)
  (G : UU l2)
  (V : Vector-Space l3 K)
  (W : Vector-Space l4 K)
  (in-V : G → type-Vector-Space K V)
  (in-W : G → type-Vector-Space K W)
  (is-free-V : is-free-Vector-Space K G V in-V)
  (is-free-W : is-free-Vector-Space K G W in-W)
  where

  linear-map-is-free-Vector-Space : linear-map-Vector-Space K V W
  linear-map-is-free-Vector-Space =
    linear-map-is-free-left-module-Ring (ring-Heyting-Field K) G V W in-V in-W
      ( is-free-V)
      ( is-free-W)
```

### The canonical isomorphism between free vector spaces

```agda
module _
  {l1 l2 l3 l4 : Level}
  (K : Heyting-Field l1)
  (G : UU l2)
  (V : Vector-Space l3 K)
  (W : Vector-Space l4 K)
  (in-V : G → type-Vector-Space K V)
  (in-W : G → type-Vector-Space K W)
  (is-free-V : is-free-Vector-Space K G V in-V)
  (is-free-W : is-free-Vector-Space K G W in-W)
  where

  is-iso-linear-map-is-free-Vector-Space :
    is-iso-Vector-Space K V W
      ( linear-map-is-free-Vector-Space K G V W in-V in-W is-free-V is-free-W)
  is-iso-linear-map-is-free-Vector-Space =
    is-iso-linear-map-is-free-left-module-Ring
      ( ring-Heyting-Field K)
      ( G)
      ( V)
      ( W)
      ( in-V)
      ( in-W)
      ( is-free-V)
      ( is-free-W)

  iso-is-free-Vector-Space : iso-Vector-Space K V W
  iso-is-free-Vector-Space =
    iso-is-free-left-module-Ring
      ( ring-Heyting-Field K)
      ( G)
      ( V)
      ( W)
      ( in-V)
      ( in-W)
      ( is-free-V)
      ( is-free-W)
```
