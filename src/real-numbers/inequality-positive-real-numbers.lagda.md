## Definition

```agda
leq-prop-ℝ⁺ : {l1 l2 : Level} → ℝ⁺ l1 → ℝ⁺ l2 → Prop (l1 ⊔ l2)
leq-prop-ℝ⁺ x y = leq-prop-ℝ (real-ℝ⁺ x) (real-ℝ⁺ y)

leq-ℝ⁺ : {l1 l2 : Level} → ℝ⁺ l1 → ℝ⁺ l2 → UU (l1 ⊔ l2)
leq-ℝ⁺ x y = type-Prop (leq-prop-ℝ⁺ x y)
```
