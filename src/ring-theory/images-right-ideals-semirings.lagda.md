## Idea

The {{#concept "image right ideal" Agda=im-right-ideal-Semiring}}
of a [right ideal](ring-theory.ideals-semirings.md) `I ⊆ R` under a [homomorphism](ring-theory.homomorphisms-semirings.md) `f : R → S` between [semirings](ring-theory.semirings.md) `R` and `S` is the [right ideal generated by](ring-theory.ideals-generated-by-families-of-elements.md) the values of `f`. In other words, the right ideal semiring `f I` is the least [right ideal](ring-theory.ideals-semirings.md) containing the element `f x` for each `x ∈ I`. Note that the [image](foundation.images.md)` of the underlying map of `f` is the [image nonunital subsemiring](ring-theory.images-nonunital-subsemirings.md), which is not necessarily closed under multiplication by an arbitrary element in `S`. For this reason, we will use the noun phrase "image right ideal" rather than the "image of the right ideal".

Since the image right ideal of a right ideal satisfies the following adjoint relation

```text
  (im f I ⊆ J) ↔ (I ⊆ J ∘ f)
```

it follows that we obtain a
[Galois connection](order-theory.galois-connections.md).

## Definitions

### The universal property of the image subsemiring of a right ideal

```agda
module _
  {l1 l2 l3 l4 : Level} (R : Semiring l1) (S : Semiring l2)
  (f : hom-Semiring R S)
  (I : right-ideal-Semiring l3 R) (K : right-ideal-Semiring l4 S)
  where

  is-image-right-ideal-Semiring : UUω
  is-image-right-ideal-Semiring =
    {l : Level} (J : right-ideal-Semiring l S) →
    leq-right-ideal-Semiring S K J ↔
    leq-right-ideal-Semiring R I (pullback-right-ideal-Semiring R S f J)
```

### The image-pullback Galois connection on right ideals

```agda
module _
  {l1 l2 : Level} (R : Semiring l1) (S : Semiring l2) (f : hom-Semiring R S)
  where

  preserves-order-im-right-ideal-Semiring :
    {l3 l4 : Level}
    (I : right-ideal-Semiring l3 R) (J : right-ideal-Semiring l4 R) →
    leq-right-ideal-Semiring R I J →
    leq-right-ideal-Semiring S
      ( im-right-ideal-Semiring R S f I)
      ( im-right-ideal-Semiring R S f J)
  preserves-order-im-right-ideal-Semiring I J H =
    inclusion-precomp-right-ideal-family-of-elements-Semiring S
      ( family-of-elements-im-right-ideal-Semiring R S f J)
      ( map-inclusion-type-subtype
        ( subset-right-ideal-Semiring R I)
        ( subset-right-ideal-Semiring R J)
        ( H))

  im-large-poset-hom-Semiring :
    hom-Large-Poset
      ( λ l → l1 ⊔ l2 ⊔ l)
      ( right-ideal-Semiring-Large-Poset R)
      ( right-ideal-Semiring-Large-Poset S)
  map-hom-Large-Preorder im-large-poset-hom-Semiring =
    im-right-ideal-Semiring R S f
  preserves-order-hom-Large-Preorder
    im-large-poset-hom-Semiring =
    preserves-order-im-right-ideal-Semiring

  image-pullback-galois-connection-right-ideal-Semiring :
    galois-connection-Large-Poset
      ( λ l → l1 ⊔ l2 ⊔ l)
      ( λ l → l)
      ( right-ideal-Semiring-Large-Poset R)
      ( right-ideal-Semiring-Large-Poset S)
  lower-adjoint-galois-connection-Large-Poset
    image-pullback-galois-connection-right-ideal-Semiring =
    im-large-poset-hom-Semiring
  upper-adjoint-galois-connection-Large-Poset
    image-pullback-galois-connection-right-ideal-Semiring =
    pullback-hom-large-poset-right-ideal-Semiring R S f
  adjoint-relation-galois-connection-Large-Poset
    image-pullback-galois-connection-right-ideal-Semiring K =
    is-image-im-right-ideal-Semiring R S f K
```

## Properties

### Any right ideal `J` of `S` has the same elements as `im f I` if and only if `J` satisfies the universal property of the image of `I` under `f`

```agda
module _
  {l1 l2 l3 l4 : Level}
  (R : Semiring l1) (S : Semiring l2) (f : hom-Semiring R S)
  (I : right-ideal-Semiring l3 R) (J : right-ideal-Semiring l4 S)
  where

  is-image-has-same-elements-right-ideal-Semiring :
    has-same-elements-right-ideal-Semiring S
      ( im-right-ideal-Semiring R S f I) J →
    is-image-right-ideal-Semiring R S f I J
  is-image-has-same-elements-right-ideal-Semiring s =
    is-lower-element-sim-galois-connection-Large-Poset
      ( right-ideal-Semiring-Large-Poset R)
      ( right-ideal-Semiring-Large-Poset S)
      ( image-pullback-galois-connection-right-ideal-Semiring R S f)
      ( I)
      ( J)
      ( sim-has-same-elements-right-ideal-Semiring S
        ( im-right-ideal-Semiring R S f I)
        ( J)
        ( s))

  has-same-elements-is-image-right-ideal-Semiring :
    is-image-right-ideal-Semiring R S f I J →
    has-same-elements-right-ideal-Semiring S
      ( im-right-ideal-Semiring R S f I) J
  has-same-elements-is-image-right-ideal-Semiring i =
    has-same-elements-sim-right-ideal-Semiring S
      ( im-right-ideal-Semiring R S f I)
      ( J)
      ( sim-is-lower-element-galois-connection-Large-Poset
        ( right-ideal-Semiring-Large-Poset R)
        ( right-ideal-Semiring-Large-Poset S)
        ( image-pullback-galois-connection-right-ideal-Semiring R S f)
        ( I)
        ( J)
        ( i))
```

### The image right ideal of the ideal `⟨x⟩` generated by a family of elements `x` under a semiring homomorphism `f` is the right ideal `⟨f ∘ x⟩` generated by the family of elements `f ∘ x`

There are three ways to state this fact:

1. The image of the right ideal `⟨x⟩` under the semiring homomorphism `f` has the same
   elements as the right ideal `⟨f ∘ x⟩`.
2. The right ideal `⟨f ∘ x⟩` satisfies the universal property of the image of the
   right ideal `⟨x⟩` under the semiring homomorphism `f`.
3. The image of the right ideal `⟨x⟩` under the semiring homomorphism `f` satisfies
   the universal property of the right ideal generated by the family of elements
   `f ∘ x`.

```agda
module _
  {l1 l2 l3 : Level}
  (R : Semiring l1) (S : Semiring l2) (f : hom-Semiring R S)
  {I : UU l3} (x : I → type-Semiring R)
  where

  forward-inclusion-image-right-ideal-family-of-elements-Semiring :
    leq-right-ideal-Semiring S
      ( im-right-ideal-Semiring R S f
        ( right-ideal-family-of-elements-Semiring R x))
      ( right-ideal-family-of-elements-Semiring S (map-hom-Semiring R S f ∘ x))
  forward-inclusion-image-right-ideal-family-of-elements-Semiring =
    leq-im-right-ideal-Semiring R S f
      ( right-ideal-family-of-elements-Semiring R x)
      ( right-ideal-family-of-elements-Semiring S (map-hom-Semiring R S f ∘ x))
      ( λ r H →
        apply-universal-property-trunc-Prop H
          ( subset-right-ideal-family-of-elements-Semiring S
            ( map-hom-Semiring R S f ∘ x)
            ( map-hom-Semiring R S f r))
          ( λ { (l , refl) →
                unit-trunc-Prop
                  ( map-right-linear-combination-family-of-elements-Semiring'
                    R S f x l ,
                    inv
                      ( preserves-ev-map-right-linear-combination-family-of-elements-Semiring'
                        R S f x l))}))

  backward-inclusion-image-right-ideal-family-of-elements-Semiring :
    leq-right-ideal-Semiring S
      ( right-ideal-family-of-elements-Semiring S (map-hom-Semiring R S f ∘ x))
      ( im-right-ideal-Semiring R S f
        ( right-ideal-family-of-elements-Semiring R x))
  backward-inclusion-image-right-ideal-family-of-elements-Semiring =
    leq-right-ideal-family-of-elements-Semiring S
      ( map-hom-Semiring R S f ∘ x)
      ( im-right-ideal-Semiring R S f
        ( right-ideal-family-of-elements-Semiring R x))
      ( λ i →
        contains-elements-im-right-ideal-Semiring R S f
          ( right-ideal-family-of-elements-Semiring R x)
          ( contains-elements-right-ideal-family-of-elements-Semiring R x i))

  compute-image-right-ideal-family-of-elements-Semiring :
    has-same-elements-right-ideal-Semiring S
      ( im-right-ideal-Semiring R S f
        ( right-ideal-family-of-elements-Semiring R x))
      ( right-ideal-family-of-elements-Semiring S (map-hom-Semiring R S f ∘ x))
  pr1 (compute-image-right-ideal-family-of-elements-Semiring s) =
    forward-inclusion-image-right-ideal-family-of-elements-Semiring s
  pr2 (compute-image-right-ideal-family-of-elements-Semiring s) =
    backward-inclusion-image-right-ideal-family-of-elements-Semiring s

  is-image-right-ideal-family-of-elements-Semiring :
    is-image-right-ideal-Semiring R S f
      ( right-ideal-family-of-elements-Semiring R x)
      ( right-ideal-family-of-elements-Semiring S (map-hom-Semiring R S f ∘ x))
  is-image-right-ideal-family-of-elements-Semiring =
    is-image-has-same-elements-right-ideal-Semiring R S f
      ( right-ideal-family-of-elements-Semiring R x)
      ( right-ideal-family-of-elements-Semiring S (map-hom-Semiring R S f ∘ x))
      ( compute-image-right-ideal-family-of-elements-Semiring)

  is-right-ideal-generated-by-family-of-elements-image-right-ideal-family-of-elements-Semiring :
    is-right-ideal-generated-by-family-of-elements-Semiring S
      ( map-hom-Semiring R S f ∘ x)
      ( im-right-ideal-Semiring R S f
        ( right-ideal-family-of-elements-Semiring R x))
  is-right-ideal-generated-by-family-of-elements-image-right-ideal-family-of-elements-Semiring =
    is-right-ideal-generated-by-family-of-elements-has-same-elements-Semiring S
      ( map-hom-Semiring R S f ∘ x)
      ( im-right-ideal-Semiring R S f
        ( right-ideal-family-of-elements-Semiring R x))
      ( inv-iff ∘ compute-image-right-ideal-family-of-elements-Semiring)
```
