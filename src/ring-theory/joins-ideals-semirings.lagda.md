## Idea

The
{{#concept "join" Disambiguation="of a family of ideals in a semiring" Agda=join-family-of-ideals-Semiring}}
of a family of [ideals](ring-theory.ideals-semirings.md) in a
[semiring](ring-theory.semirings.md) is the least ideal containing all the ideals in the
family of ideals. In other words, the join of a family of ideals is the
[ideal generated by](ring-theory.ideals-generated-by-subsets-semirings.md) the union
of the underlying subsets of the ideals in the family of ideals.

## Definitions

### The predicate of being the join of a family of ideals

```agda
module _
  {l1 l2 l3 : Level} (R : Semiring l1) {U : UU l2} (I : U → ideal-Semiring l3 R)
  where

  is-join-family-of-ideals-Semiring :
    {l4 : Level} → ideal-Semiring l4 R → UUω
  is-join-family-of-ideals-Semiring =
    is-least-upper-bound-family-of-elements-Large-Poset
      ( ideal-Semiring-Large-Poset R)
      ( I)

  inclusion-is-join-family-of-ideals-Semiring :
    {l4 l5 : Level} (J : ideal-Semiring l4 R) →
    is-join-family-of-ideals-Semiring J →
    (K : ideal-Semiring l5 R) → ((α : U) → leq-ideal-Semiring R (I α) K) →
    leq-ideal-Semiring R J K
  inclusion-is-join-family-of-ideals-Semiring J H K =
    pr1 (H K)

  contains-ideal-is-join-family-of-ideals-Semiring :
    {l4 : Level} (J : ideal-Semiring l4 R) →
    is-join-family-of-ideals-Semiring J →
    {α : U} → leq-ideal-Semiring R (I α) J
  contains-ideal-is-join-family-of-ideals-Semiring J H {α} =
    pr2 (H J) (refl-leq-ideal-Semiring R J) α
```

### The join of a family of ideals

```agda
module _
  {l1 l2 l3 : Level} (R : Semiring l1) {U : UU l2} (I : U → ideal-Semiring l3 R)
  where

  generating-subset-join-family-of-ideals-Semiring :
    subset-Semiring (l2 ⊔ l3) R
  generating-subset-join-family-of-ideals-Semiring =
    union-family-of-subtypes (λ α → subset-ideal-Semiring R (I α))

  join-family-of-ideals-Semiring :
    ideal-Semiring (l1 ⊔ l2 ⊔ l3) R
  join-family-of-ideals-Semiring =
    ideal-family-of-subsets-Semiring R (λ α → subset-ideal-Semiring R (I α))

  leq-join-family-of-ideals-Semiring :
    {l4 : Level} (K : ideal-Semiring l4 R) →
    ((α : U) → leq-ideal-Semiring R (I α) K) →
    leq-ideal-Semiring R join-family-of-ideals-Semiring K
  leq-join-family-of-ideals-Semiring K H =
    is-ideal-generated-by-family-of-subsets-ideal-family-of-subsets-Semiring R
      ( λ α → subset-ideal-Semiring R (I α))
      ( K)
      ( λ α x → H α x)

  backward-inclusion-is-join-join-family-of-ideals-Semiring :
    {l4 : Level} (K : ideal-Semiring l4 R) →
    leq-ideal-Semiring R join-family-of-ideals-Semiring K →
    (α : U) → leq-ideal-Semiring R (I α) K
  backward-inclusion-is-join-join-family-of-ideals-Semiring K H _ x p =
    H ( x)
      ( contains-subset-ideal-family-of-subsets-Semiring R
        ( λ α → subset-ideal-Semiring R (I α))
        ( x)
        ( p))

  is-join-join-family-of-ideals-Semiring :
    is-join-family-of-ideals-Semiring R I join-family-of-ideals-Semiring
  pr1 (is-join-join-family-of-ideals-Semiring K) =
    leq-join-family-of-ideals-Semiring K
  pr2 (is-join-join-family-of-ideals-Semiring K) =
    backward-inclusion-is-join-join-family-of-ideals-Semiring K

  inclusion-join-family-of-ideals-Semiring :
    {l4 : Level} (J : ideal-Semiring l4 R) →
    ((α : U) → leq-ideal-Semiring R (I α) J) →
    leq-ideal-Semiring R join-family-of-ideals-Semiring J
  inclusion-join-family-of-ideals-Semiring =
    inclusion-is-join-family-of-ideals-Semiring R I
      ( join-family-of-ideals-Semiring)
      ( is-join-join-family-of-ideals-Semiring)

  contains-ideal-join-family-of-ideals-Semiring :
    {α : U} → leq-ideal-Semiring R (I α) join-family-of-ideals-Semiring
  contains-ideal-join-family-of-ideals-Semiring =
    contains-ideal-is-join-family-of-ideals-Semiring R I
      ( join-family-of-ideals-Semiring)
      ( is-join-join-family-of-ideals-Semiring)
```

### The large suplattice of ideals in a semiring

```agda
module _
  {l1 : Level} (R : Semiring l1)
  where

  is-large-suplattice-ideal-Semiring-Large-Poset :
    is-large-suplattice-Large-Poset l1 (ideal-Semiring-Large-Poset R)
  sup-has-least-upper-bound-family-of-elements-Large-Poset
    ( is-large-suplattice-ideal-Semiring-Large-Poset I) =
    join-family-of-ideals-Semiring R I
  is-least-upper-bound-sup-has-least-upper-bound-family-of-elements-Large-Poset
    ( is-large-suplattice-ideal-Semiring-Large-Poset I) =
    is-join-join-family-of-ideals-Semiring R I

  ideal-Semiring-Large-Suplattice :
    Large-Suplattice (λ l2 → l1 ⊔ lsuc l2) (λ l2 l3 → l1 ⊔ l2 ⊔ l3) l1
  large-poset-Large-Suplattice
    ideal-Semiring-Large-Suplattice =
    ideal-Semiring-Large-Poset R
  is-large-suplattice-Large-Suplattice
    ideal-Semiring-Large-Suplattice =
    is-large-suplattice-ideal-Semiring-Large-Poset
```

## Properties

### If `I α ⊆ J α` for each `α`, then `⋁ I ⊆ ⋁ J`

```agda
module _
  {l1 l2 l3 l4 : Level} (A : Semiring l1)
  {U : UU l2}
  (I : U → ideal-Semiring l3 A)
  (J : U → ideal-Semiring l4 A)
  (H : (α : U) → leq-ideal-Semiring A (I α) (J α))
  where

  preserves-order-join-family-of-ideals-Semiring :
    leq-ideal-Semiring A
      ( join-family-of-ideals-Semiring A I)
      ( join-family-of-ideals-Semiring A J)
  preserves-order-join-family-of-ideals-Semiring =
    preserves-order-sup-Large-Suplattice
      ( ideal-Semiring-Large-Suplattice A)
      { x = I}
      { y = J}
      ( H)
```

### The operation `S ↦ (S)` preserves joins

```agda
module _
  {l1 l2 l3 : Level} (R : Semiring l1) {U : UU l2} (S : U → subset-Semiring l3 R)
  where

  is-least-upper-bound-join-ideal-subset-Semiring :
    is-least-upper-bound-family-of-elements-Large-Poset
      ( ideal-Semiring-Large-Poset R)
      ( λ α → ideal-subset-Semiring R (S α))
      ( ideal-subset-Semiring R (union-family-of-subtypes S))
  is-least-upper-bound-join-ideal-subset-Semiring =
    preserves-least-upper-bounds-ideal-subset-Semiring R S
      ( union-family-of-subtypes S)
      ( is-least-upper-bound-union-family-of-subtypes S)

  sim-preserves-join-ideal-subset-Semiring :
    sim-ideal-Semiring R
      ( ideal-subset-Semiring R (union-family-of-subtypes S))
      ( join-family-of-ideals-Semiring R (λ α → ideal-subset-Semiring R (S α)))
  sim-preserves-join-ideal-subset-Semiring =
    sim-is-least-upper-bound-family-of-elements-Large-Poset
      ( ideal-Semiring-Large-Poset R)
      { x = λ α → ideal-subset-Semiring R (S α)}
      { y = ideal-subset-Semiring R (union-family-of-subtypes S)}
      { z = join-family-of-ideals-Semiring R (λ α → ideal-subset-Semiring R (S α))}
      ( is-least-upper-bound-join-ideal-subset-Semiring)
      ( is-join-join-family-of-ideals-Semiring R
        ( λ α → ideal-subset-Semiring R (S α)))

  preserves-join-ideal-subset-Semiring :
    ideal-subset-Semiring R (union-family-of-subtypes S) ＝
    join-family-of-ideals-Semiring R (λ α → ideal-subset-Semiring R (S α))
  preserves-join-ideal-subset-Semiring =
    eq-sim-Large-Poset
      ( ideal-Semiring-Large-Poset R)
      ( ideal-subset-Semiring R (union-family-of-subtypes S))
      ( join-family-of-ideals-Semiring R (λ α → ideal-subset-Semiring R (S α)))
      ( sim-preserves-join-ideal-subset-Semiring)
```
